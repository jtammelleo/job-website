Job Match Table

This table allows you to look for jobs that might be a good match for you. For every Standard Occupation Code (which is a distinct code for every type of job), up to 10 related occupations are listed. The related occupations make use of similar skills and experience, so this should enable you to transfer to a new job with minimal additional preparation. The match rank indicates how good of a match the new occupation could be (Match Rank 1 = best match). Use the search bar to filter results, and/or click on the headers to reorder results.

Source for Job Match Data: O*Net Resource Center's Career Changers Matrix

Source for Job Openings, Wage, Education and Job Training, and Location Data: California Employment Development Department's Occupational Employment and Wage Statistics
